# AI-Impact-on-Employment-Analysis
Data analysis project exploring AI exposure, automation risk, and salary trends across job roles using Python and Power BI.

## Project Overview
This project analyzes how Artificial Intelligence is transforming employment across industries. Using Python for data cleaning and Power BI for visualization, it explores salary trends, automation risk, projected job demand, and workforce insights.

## Objectives
- Analyze salary distribution across industries
- Study AI automation risk
- Compare projected job openings
- Identify high-risk and low-risk occupations
- Understand country-wise employment trends

## Dataset
- 30,000 Job Profiles
- 8 Industries
- 8 Countries
- Salary, Experience, Required Education, Automation Risk, AI Impact Level

## Tools Used
- Python (Pandas)
- Jupyter Notebook
- MySQL
- Power BI
- Git & GitHub
